
# CarritoDeCompras Class Project Instructions

## Objective
The objective of this project is to develop a shopping cart system that allows users to add, remove, and view items in their cart, as well as calculate totals and manage user sessions.

## Self-Management Instructions
1. Allocate sufficient time for each phase of the project (design, development, testing).
2. Utilize version control to keep track of changes and collaborate effectively.
3. Regularly commit your changes with descriptive messages.
4. Perform code reviews with peers to ensure code quality.

## Technical Requirements
- Programming Language: Java
- Required Libraries: Java Swing for GUI, JUnit for testing
- Development Environment: IntelliJ IDEA or Eclipse
- Database: MySQL or any embedded database for storing user data and cart information.

## Evaluation Criteria
- Functionality: Does the application meet the specified requirements?
- Code Quality: Is the code clean, well-organized, and documented?
- User Experience: Is the user interface intuitive and user-friendly?
- Testing: Are there sufficient unit tests covering critical parts of the application?

## Methods Specifications
- `addItem(Item item)` - Adds an item to the shopping cart.
- `removeItem(Item item)` - Removes an item from the shopping cart.
- `calculateTotal()` - Returns the total price of items in the cart.
- `viewCart()` - Displays all items currently in the shopping cart.
